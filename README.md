# Exploratory Data Analysis (EDA) on Student Performance Dataset
## Project Overview
This project focuses on **Exploratory Data Analysis (EDA)** of a student performance dataset to uncover meaningful patterns, trends, and relationships between different factors affecting academic scores.
The analysis uses **Python** and popular data analysis libraries to clean, explore, visualize, and interpret the dataset.
## Objectives
- Analyze student performance data using statistical summaries
- Identify trends and hidden patterns
- Explore relationships between academic scores
- Detect correlations among variables
- Understand factors influencing student performance
- Generate visual insights for better decision-making
## Dataset
**Dataset Name:** Student Performance Dataset
**Features Included:**
- Gender
- Race/Ethnicity
- Parental Level of Education
- Lunch Type
- Test Preparation Course
- Math Score
- Reading Score
- Writing Score
## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
## Analysis Performed
### 1. Data Inspection
- View first rows
- Check dataset dimensions
- Explore column names and data types
### 2. Data Cleaning
- Missing value detection
- Duplicate row checking
### 3. Statistical Summary
- Mean
- Median
- Standard deviation
- Minimum and maximum values
### 4. Correlation Analysis
- Relationship between math, reading, and writing scores
- Heatmap visualization
### 5. Data Visualization
- Score distributions
- Boxplots for outlier detection
- Gender-wise performance comparison
- Lunch type impact analysis
- Parental education impact analysis
### 6. Feature Engineering
- Total Score calculation
- Average Score calculation
- Top-performing student identification
## Key Insights
- Strong positive correlation among math, reading, and writing scores.
- Students with completed test preparation generally score higher.
- Lunch type influences academic performance.
- Parental education level shows impact on student scores.
- Reading and writing scores are closely related.
